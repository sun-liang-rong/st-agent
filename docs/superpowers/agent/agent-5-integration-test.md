
## 缺陷分级

| 级别 | 定义 | 处理 |
|------|------|------|
| P0 阻断 | 功能完全不可用、页面白屏 | 立即修复，阻塞发布 |
| P1 严重 | 核心功能异常、数据错误 | 当天修复 |
| P2 一般 | 视觉偏差、交互不流畅 | 记录，下一轮修复 |
| P3 轻微 | 文案、对齐等细节 | 记录，有空再修 |
